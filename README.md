# 🚀 DFS PathFinder Visualization

An interactive web-based visualization tool that demonstrates how the **Depth First Search (DFS)** algorithm explores a graph.

This project is based on the famous **Romania Map problem**, a classic example used in Artificial Intelligence to understand search algorithms.

---

## 🌐 Live Demo

🔗 https://manaswiniii-26.github.io/DFS-PathFinder/

---

## ✨ Features

* 🎯 Select **Start** and **Goal** cities from the Romania map
* 🔍 Step-by-step **DFS traversal visualization** with a **clickable map**
* 📊 Real-time stats:

  * Max Depth Reached
  * Final Path Depth
  * Nodes Explored
* 🧠 Exploration log showing traversal order
* 🔄 Reset functionality to rerun the algorithm
* 🎨 Clean cyber-themed UI

---

## 🧩 About the Romania Map Problem

The **Romania Map** is a well-known problem in Artificial Intelligence, commonly used to study search strategies.

* Cities represent **nodes**
* Roads represent **edges**
* The goal is to find a path between two cities using search algorithms

This project uses that map to visually demonstrate how DFS explores deeply before backtracking.

---

## 🛠️ How It Works

* The graph is modeled using the **Romania Map dataset**
* DFS is implemented using a **stack-based approach**
* Nodes are explored deeply before backtracking
* The final path is highlighted once the goal is found
